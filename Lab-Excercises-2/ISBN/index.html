Validate ISBN Number

Rules for validation:

  1. ISBN Number can be of length 10 or 13
  2. ISBN number of length 13 can only contain numbers
  3. ISBN number of length 10 can end with an X
  4. ISBN number of length 10 is only valid if the summation of each number of ISBN multiplied by its position(from right to left) is divisible by 11
  5. ISBN number of length 13 is only valid if the summation of each number of ISBN multiplied 3(if position is odd) or 1(if position is even)(position starts from right) is divisible by 10
  6. An API call using Google Books is added at the end to find the details of the bool
